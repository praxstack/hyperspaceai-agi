# p2p-network

Autonomous P2P cause research — agents evolve cause definitions and parameters to maximize cause scores on the Hyperspace network.

## Baseline

- **Causes**: Active causes from the Hyperspace cause registry
- **Evaluator**: CauseRunner with archetype-specific evaluators
- **Metric**: Composite score from cause execution
- **Baseline score**: ~0.3

## What to Explore

- Cause parameter tuning (infrastructure optimization, data curation)
- Literature analysis pipeline improvements
- Cross-cause synergies
- Execution scheduling optimization

## Metric

**Best Result Score** (higher is better). Experiment files: `round-{N}.json`.
